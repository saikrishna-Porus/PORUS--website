# Porus Citizen Services Website

## Overview
This is the official company website for Porus Citizen Services Private Limited. The website presents the company's mission, products, services, and contact information through a modern and responsive user interface.

## Features
- Responsive design for desktop, tablet, and mobile devices
- Company introduction and about section
- Mission and vision section
- Product showcase section
- Contact section for customer inquiries
- Clean and professional user interface
- Fast loading static website

## Project Structure

```text
pous-citizen-services/
│
├── index.html      # Main webpage
├── style.css       # Website styling
├── script.js       # JavaScript functionality
├── assets/
│   ├── images/     # Images used in website
│   └── logos/      # Company logos
└── README.md
```
## Technologies Used
- HTML5
- CSS3
- JavaScript

## Installation

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in a browser.

## Local Development

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## Deployment

This website can be deployed using:
